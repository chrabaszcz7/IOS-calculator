# iOS-Style Calculator

## Description
This is a fully functional **iOS-inspired calculator** built with HTML, CSS, and JavaScript. It replicates the look and feel of the iOS calculator, including basic arithmetic operations, percentage, sign change, decimal input, and clear functions.

## Overview
The calculator features a clean and responsive design, styled to mimic the iOS calculator interface. It supports standard operations and includes an intuitive layout with buttons for numbers, operations, and utility functions.

## Features
- **iOS-style Interface:** Clean design similar to iPhone calculator.  
- **Basic Arithmetic Operations:** Addition, subtraction, multiplication, and division.  
- **Percentage Calculation:** Easily calculate percentages.  
- **Sign Toggle (+/-):** Change number sign.  
- **Decimal Input:** Supports decimal numbers.  
- **Clear (AC) and Delete:** Reset or remove last input.  
- **Responsive Design:** Works on desktop and mobile screens.

## Pages / Files
- `kalkulator.html` – Main HTML file containing the calculator layout  
- `style.css` – CSS for styling the calculator interface  
- `script.js` – JavaScript logic for calculator functionality  
- `svg/` – Folder containing icons (exit, delete, etc.)

## Usage
1. Clone or download the repository.  
2. Open `kalkulator.html` in a web browser to view the calculator.  
3. Use the calculator by clicking the buttons:  
   - Numbers for input  
   - `+`, `-`, `×`, `÷` for operations  
   - `=` to get the result  
   - `AC` to clear, `delete` to remove the last digit, `+/-` to toggle the sign, and `%` for percentages
